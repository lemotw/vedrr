# Mind Flow — Product Overview

> 供專案重新命名討論使用的功能說明文件

---

## 一句話描述

**本地優先、鍵盤驅動的桌面知識管理工具** — 以水平樹狀圖組織思維，搭配 AI 自動整理，專為 ADHD 使用者設計。

---

## 核心理念

| 理念 | 說明 |
|------|------|
| Local-first | 所有資料存在本機 SQLite，零雲端依賴 |
| Keyboard-first | Vim 風格快捷鍵操作一切，滑鼠為輔 |
| 樹狀思維 | 每個「情境」是一棵水平展開的心智樹（XMind 風格） |
| ADHD 友善 | 低摩擦操作、快速切換情境、視覺分明的節點類型 |
| AI 輔助整理 | LLM 自動重構樹結構（去重、分類、重新命名） |

---

## 功能模組

### 1. 情境管理 (Context)

- 每個情境 = 一棵獨立的知識樹 + 根節點
- 狀態：Active / Archived / Vault
- Quick Switcher（Cmd+K）：搜尋、建立、切換、歸檔情境
- 節點計數 + 最後存取時間

### 2. 樹狀編輯器 (Tree Editor)

- **水平樹佈局**：根節點在左，子節點向右展開，曲線連接線
- **四種節點類型**：
  - **Text** — 純標題節點（預設）
  - **Markdown** — 附帶 Tiptap 富文本編輯器（右側面板）
  - **Image** — 圖片縮圖 + 點擊放大燈箱
  - **File** — 附件節點，可開啟 Finder 定位檔案
- **拖放排序**：dnd-kit 實作，支援同層排序 + 跨層移動（自動防環）
- **摺疊/展開**：z 鍵或右鍵選單

### 3. Vim 風格鍵盤導航

| 操作 | 按鍵 |
|------|------|
| 同層移動 | j/k（上下兄弟） |
| 深度移動 | h/l（父/子） |
| 同層排序 | Alt+j/k |
| 跨層移動 | Alt+h/l |
| 新增子節點 | Tab |
| 新增兄弟節點 | Shift+Tab |
| 編輯標題 | Enter |
| 切換類型 | t / 1-4 |
| 搜尋節點 | Cmd+F |
| 刪除 | Delete/Backspace |
| 復原 | Cmd+Z（最多 50 步） |
| 複製/剪下/貼上 | Cmd+C/X, Ctrl+V |

### 4. 圖片處理

- **剪貼簿貼上**：Ctrl+V 自動偵測圖片，建立 Image 節點
- **手動匯入**：檔案選擇器匯入（複製到 app 資料夾）
- **縮圖 + 燈箱**：48x48 預覽，點擊全螢幕檢視
- **儲存路徑**：`~/MindFlow/files/{contextId}/{nodeId}.{ext}`

### 5. Markdown 編輯

- Tiptap 富文本編輯器（右側 480px 面板）
- 工具列：H1-H3、粗體、斜體、刪除線、程式碼、清單、引用、程式碼區塊
- 500ms 防抖自動存檔

### 6. AI Compact（AI 樹重構）

- **觸發**：選取節點 → 按 `c` 或右鍵選單
- **流程**：
  1. 讀取子樹結構 + 祖先路徑
  2. 組裝中文提示詞（去重、分類、重命名、移動）
  3. 呼叫 Anthropic / OpenAI API
  4. 解析 JSON 回應為新樹結構
  5. 顯示差異預覽（左：操作列表，右：新樹）
  6. 使用者確認後套用
- **AI 設定檔**：支援多組 profile（名稱、Provider、Model、API Key）
- **安全**：API Key 存系統 Keychain，非明文

### 7. 主題系統

- 8 個預設主題（Obsidian, Midnight, Forest, Amethyst, Mocha, Slate, Paper, Daylight）
- 自訂主題：7 色可獨立調整
- CSS Variables 驅動，即時切換

### 8. 右鍵選單

- 編輯、類型切換、摺疊/展開、複製為 Markdown
- 新增子/兄弟節點、複製/剪下/貼上
- 上下移動、AI Compact、刪除

### 9. 復原系統

- 支援：新增、刪除、標題修改、類型修改、內容修改、排序、移動
- 最多 50 步，切換情境時清空

## 技術架構摘要

```
┌─────────────────────────────────────────────┐
│ Desktop App (Tauri 2.x)                     │
│                                             │
│  ┌─ Frontend ─────────────────────────────┐ │
│  │ React 19 + TypeScript + Zustand        │ │
│  │ Tailwind v4 + Tiptap + dnd-kit         │ │
│  └────────────────────┬───────────────────┘ │
│                       │ IPC (invoke)        │
│  ┌─ Backend ──────────┴───────────────────┐ │
│  │ Rust + rusqlite + reqwest              │ │
│  │ SQLite WAL + System Keychain           │ │
│  └────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
          │
          ▼
  ~/MindFlow/data/mindflow.db
  ~/MindFlow/files/{context}/{node}.{ext}
```
